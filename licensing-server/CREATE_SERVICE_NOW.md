# Create API Service on Railway (2 minutes)

## Quick Steps

1. **Open this URL:**
   https://railway.com/project/925b1b9c-6fc0-4a19-8f80-aefca92f4ca7

2. **Click "+ New"** button (top right)

3. **Select "GitHub Repo"**

4. **Choose:** `georgekgr12/produtime-licensing-server`

5. **Click "Deploy"**

That's it! Once you click Deploy, Railway will:
- Create the service automatically
- Start building from the GitHub repo
- Deploy it

## Then tell me

Once you see the service card appear (next to Postgres), just say "done" and I'll:
- Set all environment variables via CLI
- Link the database
- Generate the public domain
- Test the endpoints

You're literally 30 seconds away from having a working server!
